# 鼠标自动化控制工具

一个基于Qt开发的电脑鼠标自动化控制软件，支持鼠标操作、键盘操作和延时功能。

## 功能特点

- 🖱️ **实时鼠标坐标显示**：实时显示当前鼠标位置坐标
- 🎯 **多种操作类型**：
  - 鼠标操作：左键单击、右键单击、左键双击、右键双击、上滚滑轮、下滚滑轮
  - 键盘操作：支持a-z字母键
  - 延时操作：可自定义延时时间（毫秒）
- 🔄 **操作重复**：支持设置每个操作的重复次数
- 📋 **操作序列**：按顺序执行添加的操作
- ⏱️ **非阻塞延时**：使用QEventLoop实现的非阻塞延时，确保UI响应流畅
- 📌 **窗口置顶**：可将窗口固定在屏幕最上层

## 技术实现

- **开发框架**：Qt 6
- **构建系统**：CMake
- **平台**：Windows
- **核心技术**：
  - QTimer实现实时坐标显示
  - QEventLoop实现非阻塞延时
  - QTableWidget管理操作序列

## 安装说明

### 方法一：从源码构建

1. 确保安装了Qt 6开发环境和CMake
2. 克隆仓库：
   ```bash
   git clone https://github.com/wdnmdQRZ/AutomatedOffice.git
   cd AutomatedOffice
   ```
3. 构建项目：
   ```bash
   mkdir build
   cd build
   cmake ..
   cmake --build .
   ```
4. 运行程序：
   ```bash
   ./AutomatedOffice.exe
   ```

### 方法二：直接运行（如果提供了可执行文件）

1. 下载发布的可执行文件
2. 双击运行 `AutomatedOffice.exe`

## 使用方法

1. **添加操作**：
   - 在"操作类型"下拉菜单中选择操作类型
   - 在"操作参数"中选择或输入具体参数
   - 在"重复次数"中设置操作执行次数
   - 点击"插入"按钮添加到操作列表

2. **执行操作**：
   - 点击"开始运行"按钮，程序将按顺序执行操作列表中的所有操作
   - 执行过程中UI仍保持响应

3. **窗口控制**：
   - 点击"固定页面"按钮可将窗口置顶/取消置顶

## 操作类型说明

| 操作类型 | 操作参数 | 说明 |
|---------|---------|------|
| 鼠标操作 | 左键单击/右键单击/左键双击/右键双击/上滚滑轮/下滚滑轮 | 执行相应的鼠标操作 |
| 键盘操作 | a-z字母 | 模拟按下相应的字母键 |
| 延时(毫秒) | 数字 | 等待指定的毫秒数 |

## 注意事项

- 本软件使用Windows API实现鼠标和键盘操作，仅支持Windows系统
- 执行操作时，请确保目标应用程序处于活动状态
- 延时操作使用非阻塞方式实现，不会影响UI响应
